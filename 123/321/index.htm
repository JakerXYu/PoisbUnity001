Oracle Database 10g Client Introduction

The Oracle Database 10g Client software enables access an Oracle Database. It provides the following installation types:

  • Instant Client: Enables you to install only the shared libraries required by Oracle Call Interface applications that use the Instant Client or Instant Client Light features. This installation type requires less disk space than the other Oracle Client installation types.

  • Administrator: Enables applications to connect to an Oracle Database on a local or remote system. It also provides administration tools for the Oracle Database.

  • Runtime: Enables applications to connect to an Oracle Database on a local or remote system.

  • Custom: Allows you to select individual components from a list of administrator and runtime components.
